# Chess
An SFML chess app where you play as White or Black against a custom engine—crisp 2D visuals, smooth animations, legal-move highlights, mouse/keyboard input, adjustable difficulty/search depth, full rule support, undo/redo, PGN import/export, time controls, and performance-tuned responsiveness for casual play and engine testing.
